How stats work
Each stat has its own XP pool and level (1–10)
You can be Intellect Lvl 6 and Warrior Lvl 2 — asymmetry is fine and expected
Overall Character Level = average of all 7 stats (round down)
This is what he shows off. A Lvl 4 is serious. A Lvl 7 is legendary at his age.

Builder Lvl 3 unlocks "startup arc" quests. Gladiator Lvl 4 unlocks olympiad boss mode.

System Rules (Anti-Burnout)
These rules keep the system healthy long-term. Without them, gamification turns into anxiety and metric-chasing within 6 weeks.
The most important rule
He must set the XP values himself for any new activity. If he doesn't believe the value is fair, the system loses meaning. You propose, he approves.
Streak protection
He earns 1 "Revival Potion" every 30 days of consistency
Can use it to protect a streak if life genuinely interrupts (illness, travel, emergency)
One "Rest Day" per week is free — no XP lost, no streak broken
Must be declared the evening before. Not a reactive escape hatch.
After a miss: just restart. No penalty XP.
The system never punishes — it only rewards. Missing is zero, not negative.
Progress review schedule
Daily: 2-min log (done vs not done on 3 tasks)
Sunday: 20-min weekly review — tally XP, pick next week's side quests, update stat bars
Monthly: System recalibration — are XP values still fair? Any new quests to add? Coins to spend?
Quarterly: Arc review — which main quests advanced? Which boss fights are coming?
The 3 deadly sins of gamification
Never compare his stats to anyone else's
Only compare to his own past. This system has no leaderboard.
Never make the system feel like another obligation
If logging feels like homework, simplify it. The system serves him, not the other way around.
Never chase XP over real outcomes
If he finds shortcuts that earn XP without learning, fix the XP value — don't scold him.
